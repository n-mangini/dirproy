# SPLT-001: Crear Evento

**Como** Usuario Registrado  
**Quiero** crear un nuevo evento con un nombre  
**Para que** pueda empezar a organizar los gastos del grupo.

---

## 🎯 Criterios de Aceptación (Verificables)

1. El sistema debe validar que el nombre del evento no sea solo espacios en blanco.
2. El límite de caracteres para el nombre del evento es de 50.
3. Al guardar, el usuario debe ver inmediatamente la pantalla del evento creado.
4. El evento debe mostrar por defecto al creador como el primer participante.

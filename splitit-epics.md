# Épicas del producto - SplitIt

## 1. Gestión de Eventos
Permite crear, editar eventos y gestionar participantes.

## 2. Acceso con Registro
Permite acceder mediante login para poder crear los splits/eventos.

## 3. Acceso sin Registro
Permite acceder mediante magic links sin necesidad de login.

## 4. Gestión de Gastos
Permite registrar, editar y eliminar gastos dentro de un evento.

## 5. Cálculo de Balances
Permite calcular automáticamente quién le debe a quién.

## 6. Visualización de Cuentas
Permite visualizar balances individuales y del grupo.

## 7. Multi-moneda
Permite convertir gastos a una moneda base mediante API externa.
